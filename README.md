# Drupal 7 — Кастомные модули

Данные модули разработаны с помощью искусственного интеллекта [Claude](https://claude.ai) от Anthropic. Каждый модуль решает конкретную задачу и полностью интегрируется в стандартную архитектуру Drupal 7: хуки, блоки, страницы настроек через админку.

---

## Модули

### custom_cookies

Модуль отображает плашку согласия с cookie для соответствия требованиям GDPR. Текст уведомления, ссылка на политику конфиденциальности и внешний вид кнопки настраиваются через административный интерфейс без правки кода. При нажатии кнопки в браузере устанавливается cookie, и плашка больше не показывается при повторных визитах.

### custom_newyear

Модуль добавляет на сайт новогоднее оформление: анимированную гирлянду из цветных лампочек и эффект падающего снега. Через административную панель можно включать и отключать каждый эффект независимо, настраивать количество, размер и цвет снежинок, задавать фон и отступы для гирлянды. Гирлянда реализована как Drupal-блок и размещается в любом регионе темы.

### token_bulk_editor

Модуль добавляет единую страницу для массового редактирования всех кастомных токенов модуля `token_custom`. Вместо того чтобы открывать каждый токен отдельно, можно изменить значения и форматы текста сразу для всех токенов и сохранить одной кнопкой. Токены на странице сгруппированы по типу.
